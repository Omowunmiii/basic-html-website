# Basic HTML Website

This project is a simple website built using only HTML. It demonstrates basic page structure with headings, links, and multiple pages.

## How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Double-click any `.html` file (e.g., `index.html`) to open it in a web browser.

## Project Files

- `index.html` — Home page  
- `articles.html` — Articles page  
- `contact.html` — Contact page  
- `projects.html` — Projects page
